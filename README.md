# Rust Terminal Text Calcualtor 

A command-line calculator written in Rust that reads and evaluates math expressions typed as text (e.g. 3 + 4 * 2), using standard operator sequence and parentheses.

## Instructions for Build and Use

Steps to build and/or run the software:

1. Install Rust and Cargo if you don't already have them, via rustup.
2. Clone/download the repository
3. From the project's root directory, run cargo build to compile the project

Instructions for using the software:

1. Run cargo run to start the prompt feature
2. Type a math prompt such as 2+4 or (2/3)*4
3. Type quit to exit 

## Development Environment

To recreate the development environment, you need the following software and/or libraries with the specified versions:

* Rustc - 1.9 or later 
* Cargo - bundled with rust 
* Standard Rust library
